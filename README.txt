Projet contenant des tests sans SUT en JUnit Jupiter. Chaque test contient une simple assertion sur un calcul basé sur deux variables.
Ce projet est composé de 2 classes avec un mix de tests OK et de tests en failure.

Calculator.java : - addSuccess(displayName = "Addition") : OK
		  - multFailure : assertTrue non validé

CalcFailure.java : - subFailure(displayName = "Substraction")
